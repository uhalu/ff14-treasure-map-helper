# FF14 宝の地図ヘルパー

FF14 トレジャーハントで解読した「宝の地図」のスクリーンショットから、場所（エリア名と座標）を
すぐに調べられる非公式のファンメイドツールです。地図の一覧と目視で見比べて探す手間を省きます。

**https://uhalu.github.io/ff14-treasure-map-helper/** （スマホ対応・インストール不要）

*Available in 日本語 / English / Deutsch / Français / 한국어 / 简体中文 — use the language
selector at the bottom of the screen.*

## 使い方

1. スマホのブラウザで上記 URL を開く
2. モニタに表示された宝の地図にカメラを向け、ガイド枠に合わせる
3. 見つかると振動して「エリア名 (X:xx.x, Y:yy.y)」が表示される

PC のブラウザでは [PC用モード](https://uhalu.github.io/ff14-treasure-map-helper/desktop.html)
も使えます。地図ウィンドウを Win+Shift+S で切り取って Ctrl+V で貼り付ける（または画像を
ドロップする）と照合され、結果の「エリア名 (X:xx.x, Y:yy.y)」がクリップボードへコピーされます。

表示言語は画面下のセレクタで切り替えられます（初回はブラウザの言語から自動選択）。
エリア名はゲーム内の正式名称で表示され、ゾーン名の読み取り（OCR）も選択した言語で行われます。

## 収録範囲

宝の地図 G1〜G18・隠された地図・謎めいた地図・ソーン朝時代の地図（全 806 スポット）。

## 安心してご利用頂くために

- **画像はすべて端末内（ブラウザ内）で処理**します。カメラ映像も、PC用モードで貼り付け・
  ドロップしたスクリーンショットも、外部へ送信する処理はありません。本サイトは静的ページのみで、
  画像を受け取るサーバーがそもそも存在しません。照合データベースの読み込みが済めば、
  通信を切った状態でも照合できます。
- 利用状況の把握のため、Cookie を使わないアクセス解析
  （[GoatCounter](https://www.goatcounter.com/)）のみ利用しています。本アプリのコードに
  画像・照合結果・入力内容を外部へ送信する処理はありません。
- **ゲームクライアントには一切触れません**。メモリ読み取り・操作の自動化・MOD の類ではなく、
  ゲームの外で、手元の画像だけを照合します。
- **非商用の個人ツール**です。収益化・広告・寄付の受付はありません。
- リポジトリと配布物に**ゲームの画像素材は含まれません**。照合用データベースに入っているのは、
  地図ごとの数値特徴量（ハッシュ値と 16×16 の明暗傾向。元画像の復元や代替表示に使える
  ものではありません）と、エリア名・座標などの公知情報だけです。
- FINAL FANTASY XIV は株式会社スクウェア・エニックスの著作物です。本ツールは非公式であり、
  スクウェア・エニックスとは一切関係ありません。

## 開発者向け

```bash
# 要 Node 20+
cd web && npm install && npm test && npm run build   # → web/dist/
```

照合エンジンは `web/src/matcher/`。照合用データベース `data/maps.json` はビルド時に
`web/public/` へコピーされます。`web/test/fixtures` のゴールデンデータで特徴量抽出の
回帰を検証しています。
